## 🏆 AWS Hackathon — Top 3 Finalist

Built and submitted for an **AWS-judged hackathon**, earning a **Top 3 finish** evaluated on cloud-native architecture, security design, AI integration, and real-world applicability. All inference runs through **AWS Bedrock** within a secure VPC — no third-party AI providers.

---

##  What It Does

1. **Upload resume** → AWS Textract parses and extracts skills
2. **Enter target role** → Claude Sonnet 4.5 analyzes skill gaps
3. **Get a 28-day roadmap** → Weekly modules with daily 15-minute sprints
4. **Get a professional resume** → AI-generated and tailored to target role
5. **Track progress** → Milestones, portfolio projects, interview prep
